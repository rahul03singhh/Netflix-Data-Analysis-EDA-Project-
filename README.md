# Netflix-Data-Analysis-EDA-Project-
A comprehensive exploratory data analysis of the Netflix catalog using Python. This project uncovers trends in content growth, genre popularity, and global distribution patterns.

I’ve always been curious about how Netflix chooses its content. Does it prefer movies over shows? Is it still just an American platform, or has it truly gone global? I decided to dive into the Netflix Titles dataset to find out.

In this project, I took the raw data, cleaned up the messy parts (like missing directors and inconsistent ratings), and built some visualizations to see the "big picture" of Netflix’s strategy.

What I found:

The "TV Show" Surge: While Netflix started as a movie-heavy service, there’s a massive spike in TV shows starting around 2015. They clearly want us binge-watching!

Global Flavors: The US is the biggest producer, but India is a powerhouse in the movie category. It was cool to see how much international content is actually being pushed.

Adult-Oriented: Most of the catalog is TV-MA or TV-14. Netflix isn't just for kids; they are heavily targeting mature audiences.

Drama is King: No matter the country, Drama and Comedy are the most consistent genres.

How I built It:

I used Python for the whole thing.

Pandas was my best friend for cleaning the data (especially splitting those genre strings!).

Seaborn and Matplotlib helped me turn rows of numbers into charts that actually make sense.

To check it out:

Make sure you have Python installed.

Grab the netflix_titles.csv and the .ipynb file from this repo.

Run it in Jupyter—I’ve left comments throughout the code to explain my logic.
